# Calculator

A fully functional, clean calculator website built as an assignment for the KMEA Coding Club MERN Stack Internship framework. This project focuses on handling user input, managing states, and executing mathematical operations using JavaScript.

## ✨ Features
- **Basic Math:** Performs addition, subtraction, multiplication, and division.
- **Display Management:** Starts at `0` and updates dynamically as buttons are clicked.
- **Clear Utility (C):** Flushes the current display back to `0`.
- **Backspace (⌫):** Removes the last entered character safely without breaking the display layout.

---

## 🛠️ Tech Stack & Languages Used
* **HTML5:** Defines the display screen and the layout grid of buttons.
* **CSS3:** Handles the dark-themed UI, rounded buttons, and orange highlight coloring for operator actions.
* **JavaScript (Vanilla JS):** Drives the calculator engine, handles button click events, and computes results using the `eval()` evaluation logic.

---

## 📂 File Structure
* `index.html` — The core structure and layout configuration.
* `style.css` — The visual styling rules and button dimensions.
* `script.js` — The logic engine for display updates and calculations.
