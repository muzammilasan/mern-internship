# ToDo List Website

A simple and interactive ToDo List website built as part of my training with the KMEA Coding Club. This project shows how I use JavaScript to make a webpage move and react to user clicks.

## ✨ Features
- **Add Tasks:** Type a task and click "Add" to put it on the list.
- **Mark as Done:** Click "Done" to change the task color to green.
- **Edit Tasks:** Click "Edit" to fix or change a task you already wrote.
- **Delete Tasks:** Click "Delete" to remove a single task from the screen.
- **Clear Everything:** Click "Clear All" to wipe the entire list clean.

---

## 🛠️ Languages Used
* **HTML5:** Used to build the structure of the website (buttons, boxes, headings).
* **JavaScript (Vanilla JS):** Used to add the logic (making the buttons actually work, creating new list items, and deleting tasks).

---

## 📂 Project Files
* `index.html` — The main webpage structure.
* `script.js` — The JavaScript code that runs the app logic.
