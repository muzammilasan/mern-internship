# MERN Stack Web Development Internship

This repository contains the code, assignments, and full-stack web applications developed during the 4-week **AI-Enhanced MERN Stack Web Development Internship**. 

## 📋 Program Overview
* **Course Provider:** Estelle Darcy
* **Host Community:** KMEA Coding Club (CC)
* **Duration:** 4 Weeks (Started: May 18, 2026)
* **Mode:** Online / Virtual Training

---

## 🛠️ Tech Stack & Skills
This program bridges the gap between static web design and dynamic full-stack application development using the **MERN** architecture:

* **M**ongoDB: NoSQL Cloud Database (using MongoDB Atlas)
* **E**xpress.js: Minimalist backend web framework for Node.js
* **R**eact.js: Component-based frontend library for dynamic UI
* **N**ode.js: JavaScript runtime environment for building scalable backend servers

### Key Concepts Covered:
- **JavaScript (ES6+):** Functions, variables, asynchronous operations, and logic formatting.
- **REST APIs:** Designing, creating, and consuming API endpoints to connect frontend and backend.
- **Authentication:** Implementing secure user registration, login, and authorization using **JWT (JSON Web Tokens)**.
- **Git & GitHub:** Version control, branching, committing, and remote repository tracking.
- **Deployment:** Cloud hosting and launching live full-stack web applications.
